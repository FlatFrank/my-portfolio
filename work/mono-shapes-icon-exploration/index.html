Mono Shapes — Icon exploration

A small icon set experiment exploring weight, geometry, and legibility.

CategoryIllustration

Overview

A small icon set experiment exploring geometry, optical balance, and legibility at 16–24px.

What I explored

  • Stroke weight and corner radius

  • Negative space alignment

  • Filled vs outlined variants

  • Element

Sample notes

The goal was consistency over uniqueness. Each icon is drawn on the same grid, then optically adjusted so diagonals and curves don’t feel lighter than vertical strokes.
